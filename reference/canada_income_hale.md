# Canada income quintile HALE data

Health-Adjusted Life Expectancy for Canada, stratified by household
income quintile. For use in Canadian DCEA analyses (CADTH workflow).

## Usage

``` r
canada_income_hale
```

## Format

A tibble with 5 rows and 9 variables analogous to
[`england_imd_hale`](https://heorlytics.github.io/dceasimR/reference/england_imd_hale.md)
but with income-based stratification.

## Source

Statistics Canada, Health-Adjusted Life Expectancy by income quintile.
